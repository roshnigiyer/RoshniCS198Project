Technology

Development and API Resources

Google Hangouts API

Netflix&Chat implements the Google Hangouts API to allow users to chat and watch videos together. Implementing the Hangouts API allows the app to include the functionality of Google Accounts without having a seperate implementation.

Javascript

Earlier in the project, we spent a lot of time attempting to use Javascript to accomplish window resizing before discovering the reason all of our attempts were failing was because Google Chrome itself recently disabled Javascript resizing to prevent malware from taking over the browser.

CSS

Instead of using Javascript to resize the window, we've played around with injecting CSS to control the size of all page elements. This process works, but isn't efficent or well-developted because Netflix changes their content structure often (ie. whenever they update the show banners, all the div classes change)

HTML

Within our Chrome Extension, we used HTML in tandem with CSS to style our extension layout and pages. We also used HTML to develop and style content on our website.
